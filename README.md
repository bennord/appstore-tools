# appstore-tools

Tools for interacting with the AppStore Connect api

### Prereqs

- python (use pyenv to install `v3.9.*`)
- pipenv (`pip install pipenv`)
- black (set this as the python formatter in whatever code editor you use)

### Install

Install dependencies

```
pipenv install --dev
```

### Run

```
pipenv shell
python run.py [args...]
```
